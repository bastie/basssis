Welcome and hello to Basssis!

Basssis is Bastie's small server side includes server.

This is a simple SSI server written in Rust. Why Rust? Because I do not need too much dependencies on my target system to build and compile. Why new server? Because I need not so much but I need some more.

Supported features

It supports the GET command from HTTP.

It supports the #exec command from SSi. On (Free)BSD and other systems for example you see the OS nformation with command /usr/bin/uname -a:

Configuration

The server need a program named "get_IP", with x flag on BSD. This PRG need to print the binding IP, not port, on CLI.
